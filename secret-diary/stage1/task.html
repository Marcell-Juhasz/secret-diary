Description

In the first stage, you'll implement the basic diary functionality. Users should be able to enter a note they want and save it with a button. After this operation, the saved text should appear in the diary.

Objectives

1. Create the following components of the user interface with the appropriate features:

  • An Activity named MainActivity;
  • A EditText field with the id attribute set to etNewWriting and the hint attribute set to Dear Diary...;
  • A Button with the id of btnSave and the text Save;
  • A TextView with the id of tvDiary that shows the saved text after a click on the save button.

2. Add the following functionalities to the application:

  • When the Save button is clicked, the diary should display exactly the same text as the user entered;
  • If the diary already contains something, overwrite it;
  • After every saving, clear the input field.

3. Handle the exceptional cases:

  • When the Save button is clicked, and the input is either empty or contains only whitespace characters, the input should not be saved, and a Toast message should warn users with the following text: Empty or blank input cannot be saved
