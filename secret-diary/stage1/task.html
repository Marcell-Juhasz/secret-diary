Description

In the first stage, you should implement the basic functionality of the diary. The user should be able to enter some text they want and save it with a button. After this operation, the saved text should appear in the diary.

Objectives

1. Create the following components of the user interface with the appropriate features:

  • An EditText field with an id attribute set to "etNewWriting" and a hint attribute set to "Dear Diary...".
  • A Button with the id "btnSave" and the text "Save".
  • A TextView with the id "tvDiary" that shows the saved text after a click performed on the save button.

2. Add the following functionalities to the application:

  • When the Save button is clicked, the diary should display exactly the same text as the user entered.
  • If the diary already contains some text, overwrite it again and again.
  • After every saving operation, the input field should be cleared.
