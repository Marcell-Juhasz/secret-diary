Description

You have created a lovely diary so far. Users can add as many notes as they want and remove the last one. The application, however, should also be able to save the data; that is what you must implement in this stage. When the application is closed by users, all the diary text data should be saved, including the date and time notations. Users should see the diary precisely as it was when they restart the application. The Save and Undo buttons should work as before.

Objectives

  1. Save the diary into SharedPreferences with the name PREF_DIARY and the key KEY_DIARY_TEXT to display it once the application closes. Save the diary after each Save and Undo action;
  2. When the user restarts the app, load the saved diary automatically;
  3. The two buttons should work like in the previous stage; the same applies to the notes loaded from the storage.
