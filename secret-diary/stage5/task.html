Description

The diary is ready to use, well done! But since we may write down our secrets, do not want others to access and read them. So, the last step is to protect the diary with a password that only the user knows.

To do this, you have to create a new Activity for the PIN, which is shown when the application is launched. If the user correctly enters the secret number, then tap the Log In button, they can access the diary and work with it like in the previous stage. If the PIN is incorrect, let the user know by showing an error message.

Objectives

1. Create a new Activity named "LoginActivity" with the following components and set it to be the launcher activity:

  • A TextView with id "tvTitle" and text "Secret Diary"
  • An EditText with id attribute set to "etPin" that only accepts numeric input. Also, a hint "Enter PIN" should inform the user about the role of the input field.
  • A Button with "btnLogIn" id and "Log In" text property

2. If the user presses the Log In button and the entered PIN is correct (PIN: 1234), launch the diary Activity you already implemented in the previous stages. Otherwise set the error property of the input text field to "Wrong PIN!"
